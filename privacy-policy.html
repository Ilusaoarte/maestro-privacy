Política de Privacidade

Maestro — Controlador de Áudio Profissional
Última atualização: 7 de junho de 2026

1. Introdução

O Maestro ("nós", "nosso" ou "app") é um aplicativo de controle de áudio profissional desenvolvido por Everton Machado. Esta política explica como tratamos seus dados.

Resumo: Playlists, arquivos de áudio, gravações e dados do Maestro Vision permanecem no seu dispositivo. O app pode usar Firebase para login/licença Pro, Firebase Crashlytics para relatórios técnicos de erro e RevenueCat/App Store para compras e restauração de licença.

2. Dados tratados pelo app

A versão pública do Maestro pode tratar os seguintes dados, sempre limitados à finalidade do recurso usado:

  • Conta Pro: quando você faz login, o Firebase Authentication trata seu e-mail, identificador de usuário e estado de autenticação.
  • Licença Pro: o Firebase pode armazenar o identificador da conta e o status de licença para liberar recursos pagos.
  • Compras: RevenueCat e App Store podem tratar histórico de compra, identificador de usuário do app e status de licença para compra, restauração e suporte.
  • Relatórios de erro: em versões de produção, o Firebase Crashlytics pode receber dados técnicos de falhas, como versão do app, sistema operacional, stack trace e informações de diagnóstico.
  • Dados locais: playlists, configurações, arquivos de áudio, gravações e calibrações ficam armazenados no dispositivo, salvo quando você decide exportar ou compartilhar.

3. Dados que o app NÃO coleta

  • Não vendemos dados pessoais.
  • Não exibimos anúncios.
  • Não usamos dados de câmera para identificar pessoas.
  • Não rastreamos sua localização geográfica para publicidade ou perfilamento.
  • Não enviamos seus arquivos de áudio, playlists ou gravações para servidores do Maestro.
  • Não compartilhamos dados com terceiros para marketing.

4. Permissões do dispositivo

  • Câmera: usada para detecção de gestos em tempo real (Maestro Vision). As imagens são processadas localmente no dispositivo e nunca são gravadas, armazenadas ou transmitidas.
  • Microfone: permite gravar cues de áudio diretamente no app. O áudio gravado é salvo apenas no armazenamento local do dispositivo.
  • Biblioteca de mídia/Fotos: necessário para navegar e importar arquivos de áudio (MP3, WAV, AAC, FLAC, etc.) armazenados no dispositivo, e para exportar backups.
  • Biblioteca de música: permite importar faixas de áudio da biblioteca do dispositivo para uso como cues.

5. Armazenamento de dados

Playlists, configurações, arquivos de áudio e dados de calibração são armazenados no dispositivo. Dados de autenticação, compra e licença podem ser mantidos pelo Firebase, RevenueCat e App Store para validar o acesso Pro.

6. Segurança

Relatórios de diagnóstico são enviados apenas para análise técnica de falhas do app. Arquivos de áudio, playlists, gravações e dados de câmera permanecem no dispositivo, salvo quando você decide exportar ou compartilhar.

7. Crianças

O Maestro não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de crianças.

8. Alterações nesta política

Caso esta política seja atualizada, a nova versão será publicada nesta mesma página com a data de atualização revisada.
